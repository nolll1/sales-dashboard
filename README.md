# 🛒 Ecommerce Sales Dashboard (Power BI)

## 📊 Overview
This project showcases an interactive Ecommerce Sales Dashboard built using Microsoft Power BI to analyze revenue performance, customer behavior, product demand, and geographical distribution.

The dashboard enables data-driven decision-making by providing insights into sales trends, top-performing products, and customer spending patterns.

## 🧠 Data Modeling

- Implemented a **star schema** for efficient querying and reporting
- Fact Table:
  - Sales transactions (revenue, quantity, pricing)
- Dimension Tables:
  - Customer
  - Product
  - Date
  - Geography

---

## ⚙️ Tech Stack

- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Data Modeling (relationships, schema design)

---

## 📌 Key Insights

- Revenue and quantity both increased by **16.1%**, indicating strong growth
- Majority of revenue driven by a few high-performing countries
- Product demand is concentrated in top categories (Ct, Cans)
- Customer spending is unevenly distributed (top customers contribute significantly)
